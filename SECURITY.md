# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public GitHub issue
2. Email the maintainers or use [GitHub's private vulnerability reporting](https://github.com/assistant-ui/modelpedia/security/advisories/new)
3. Include a description of the vulnerability and steps to reproduce

We will respond within 48 hours and work with you to resolve the issue.

## Scope

This project is a data catalog and does not handle user credentials or sensitive data directly. However, we take security seriously for:

- The web application (modelpedia.dev)
- The REST API (api.modelpedia.dev)
- Supply chain security of the npm package
